# Grocery Store Management System (Python)

A desktop-based Grocery Store Management System built using Python, Tkinter, and SQLite.

## Features
- Login authentication
- Add / View / Update / Delete products
- Low stock alerts
- Product search
- Billing / POS system
- Persistent database using SQLite

## Technologies Used
- Python
- Tkinter
- SQLite

## How to Run
1. Install Python
2. Run `login.py`
3. Login with:
   - Username: admin
   - Password: 1234
